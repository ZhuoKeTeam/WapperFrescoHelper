# WapperFrescoHelper
Fresco 使用封装

如何使用：

1.在根项目下的 build.gradle 添加如下代码：
```
allprojects {  
    repositories {  
        ...  
        maven { url 'https://jitpack.io' }  
    }  
}  
```

  
2.在项目下的 build.gradle 添加依赖：
```
dependencies {  
	        compile 'com.github.ZhuoKeTeam:WapperFrescoHelper:1.0.0'      
}  
```
